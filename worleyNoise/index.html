What's the matter about slides and checkbox?

  • Seed Points are the number of random initial points.
  • n-th distance is the position of the distance that the algorithm takes to the seed points, in ascending order. For example, if n=1, we have a Voronoi Diagram.
  • Palette in color mode, is a choice of a RGB colors permutation (3!=6).
  • Bright is the percentage apply to the RGB value. Example if bright equal a 50%, max value= 50% of 255=127.
  • Ratio if the distance to one pixel is out of range fixed by the ratio, this pixel will be painted black (or white in dual mode).
  • Dual Reverse the order of RGB values or gray scale.
  • Color Switch color mode to gray scale mode and vice versa.
Animation-> For each frame, seed points move a random value wich causes the movement.
